## 👤 Hesap Türleri

### Yük Sahibi (Cargo Owner)
- Yük ilanı verebilir
- Gelen teklifleri değerlendirebilir
- Nakliyecilerle mesajlaşabilir
- Taşıma sürecini takip edebilir

### Nakliyeci (Carrier)
- Yük ilanlarını görüntüleyebilir
- İlanlara teklif verebilir
- Yük sahipleriyle iletişim kurabilir
- Araç bilgilerini yönetebilir

## 📝 Yük Sahibi İçin Kullanım

### 1. Hesap Oluşturma
```
1. Ana sayfada "Kayıt Ol" tıklayın
2. "Yük Sahibi" seçin
3. Bilgileri doldurun:
   - Kullanıcı adı
   - Email
   - Şifre
   - Ad Soyad
   - Telefon
   - Şirket adı (opsiyonel)
   - Adres
4. "Hesap Oluştur" tıklayın
```

### 2. Yük İlanı Verme
```
1. Dashboard'a giriş yapın
2. "Yeni İlan Ver" butonuna tıklayın
3. İlan bilgilerini doldurun:
   - İlan başlığı
   - Açıklama
   - Yük türü
   - Ağırlık
   - Alış şehri/adresi
   - Teslim şehri/adresi
   - Alış tarihi
   - Bütçe bilgileri
4. "İlan Oluştur" tıklayın
```

### 3. Teklifleri Değerlendirme
```
1. Dashboard'da "Gelen Teklifler" bölümünü kontrol edin
2. Her teklif için:
   - Nakliyeci bilgilerini inceleyin
   - Teklif miktarını değerlendirin
   - Nakliyeci profilini görüntüleyin
3. Uygun teklifi "Kabul Et" ile onaylayın
4. Diğer teklifleri "Reddet" ile kapatın
```

### 4. Mesajlaşma
```
1. Kabul edilen teklif sonrası otomatik chat açılır
2. "Mesajlar" bölümünden erişebilirsiniz
3. Nakliyeci ile detayları konuşun:
   - Alış zamanı
   - Teslim zamanı
   - Özel talepler
   - İletişim bilgileri
```

## 🚛 Nakliyeci İçin Kullanım

### 1. Hesap Oluşturma
```
1. Ana sayfada "Kayıt Ol" tıklayın
2. "Nakliyeci" seçin
3. Bilgileri doldurun:
   - Temel bilgiler (ad, email, telefon)
   - Şirket bilgileri
   - Araç bilgileri:
     * Araç türü
     * Kapasite
     * Plaka
4. "Hesap Oluştur" tıklayın
```

### 2. İlan Arama ve Filtreleme
```
1. "Yük İlanları" sayfasına gidin
2. Filtreleri kullanın:
   - Yük türü
   - Alış/teslim şehri
   - Ağırlık aralığı
   - Tarih aralığı
   - Bütçe aralığı
3. Uygun ilanları listeleyin
4. İlan detaylarını inceleyin
```

### 3. Teklif Verme
```
1. Uygun bir ilan bulun
2. "Teklif Ver" butonuna tıklayın
3. Teklif bilgilerini girin:
   - Teklif miktarı
   - Para birimi
   - Açıklama/notlar
   - Tahmini süre
4. "Teklif Gönder" tıklayın
```

### 4. Teklif Takibi
```
1. Dashboard'da "Verdiğim Teklifler" bölümünü kontrol edin
2. Teklif durumları:
   - Beklemede: Henüz yanıtlanmamış
   - Kabul Edildi: Yük sahibi onayladı
   - Reddedildi: Teklif kabul edilmedi
3. Kabul edilen teklifler için mesajlaşma başlar
```

## 💬 Mesajlaşma Sistemi

### Mesaj Gönderme
```
1. Dashboard'da "Mesajlar" bölümüne gidin
2. Aktif konuşmaları görüntüleyin
3. Mesaj yazın ve "Gönder" tıklayın
4. Dosya ekleme özelliği (gelecek güncellemede)
```

### Mesaj Bildirimleri
```
- Yeni mesajlar dashboard'da görünür
- Email bildirimleri (ayarlanabilir)
- Browser bildirimleri (gelecek güncellemede)
```

## 🔧 Profil Yönetimi

### Profil Güncelleme
```
1. Sağ üst köşedeki profil menüsüne tıklayın
2. "Profil Ayarları" seçin
3. Güncellemek istediğiniz bilgileri değiştirin:
   - Kişisel bilgiler
   - İletişim bilgileri
   - Şirket bilgileri
   - Araç bilgileri (nakliyeciler için)
4. "Kaydet" tıklayın
```

### Şifre Değiştirme
```
1. Profil ayarlarında "Şifre Değiştir" bölümüne gidin
2. Mevcut şifrenizi girin
3. Yeni şifrenizi girin
4. Şifre tekrarını girin
5. "Şifre Güncelle" tıklayın
```
